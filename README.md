# Nest Store API

Necesitas tener [node js](https://nodejs.org/en), [Docker](https://www.docker.com/), [Nest js](https://docs.nestjs.com/)

Clonar pryecto

```
yarn install
```

Cloar `.env.example` y renombrarlo a `.env`

Cambiar las variables de entorno

Levantar base de datos
```
docker-compose up -d
```
